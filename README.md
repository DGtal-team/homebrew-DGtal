# DGtal / Hombrew

This repository is a [tap](https://docs.brew.sh/Taps) for [homebrew](http://brew.sh) to build Dgtal tools on MacOS.

Simply run:

    brew tap DGtal-team/DGtal
    brew install dgtal
    
(`brew options dgtal` for more build options)
